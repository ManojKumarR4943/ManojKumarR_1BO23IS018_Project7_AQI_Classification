# Air Quality Index Classification

## 📌 Description

This project uses Machine Learning techniques to classify air quality levels based on pollutant data. It applies Principal Component Analysis (PCA) for dimensionality reduction and Nearest Centroid Classifier (NCC) for classification.

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn

---

## ⚙️ Features

* Data preprocessing and cleaning
* Dimensionality reduction using PCA
* Classification using Nearest Centroid Classifier
* Accuracy evaluation
* Data visualization using graphs

---

## 🏗️ Architecture

Data Collection → Preprocessing → PCA → NCC → Prediction → Visualization

---

## 📊 Output

* Accuracy: ~91%
* Graphs:

  * Heatmap
  * Distribution Plot

---

## ▶️ How to Run

1. Install required libraries:
   pip install numpy pandas scikit-learn matplotlib seaborn

2. Run the program:
   python main.py

---

## 📁 Project Files

* main.py → Source code
* data.csv → Dataset
* report.docx → Project report
* README.md → Documentation

---

## ✅ Conclusion

The project successfully classifies air quality levels with good accuracy. PCA helps in reducing data complexity, and NCC provides efficient classification.
